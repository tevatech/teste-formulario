# Formulário de Contato Responsivo

Projeto de um formulário de contato responsivo desenvolvido com HTML, CSS, Bootstrap e JavaScript.  
O formulário valida os campos, exibe mensagem de sucesso e armazena os dados no navegador.

---

## Funcionalidades

- Layout responsivo com Bootstrap
- Campos com label flutuante
- Validação dos campos com JavaScript
- Mensagem de sucesso ao enviar o formulário
- Armazenamento dos dados usando LocalStorage

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
